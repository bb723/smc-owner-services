## Use it three ways
1. **PDF to hand out:** open `index.html` in any browser, click "Print / Save as PDF". The
   print stylesheet hides the button and keeps colors.
2. **GitHub Pages link:** push this folder to a repo (or `/docs`) and enable Pages. Mirrors the
   `cap-rate-calculator` setup.
3. **Local link:** just send someone the opened file.
